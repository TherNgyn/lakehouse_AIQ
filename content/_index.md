# Building Lakehouse Architecture for AIQ Index Evaluation

### Overview
In today's era of digitalization and Big Data, efficiently collecting, storing, processing, and analyzing data has become a key factor determining the competitive advantage of organizations and businesses. This workshop will guide you through building a comprehensive Lakehouse architecture on AWS to collect, process, and analyze IoT sensor data to evaluate the AIQ (AI Quality) index.
